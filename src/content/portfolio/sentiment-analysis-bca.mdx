# Sentiment Analysis BCA Apps

## Project Overview
A comprehensive sentiment analysis system designed to analyze user reviews and feedback for the BCA (Bank Central Asia) mobile application using state-of-the-art deep learning techniques.

## Problem Statement
Understanding customer sentiment from app reviews is crucial for:
- Identifying pain points and bugs
- Improving user experience
- Prioritizing feature development
- Measuring customer satisfaction

## Technical Implementation

### Data Collection & Preprocessing
- **Dataset**: 50,000+ user reviews from Google Play Store
- **Preprocessing Steps**:
  - Text cleaning and normalization
  - Emoji and emoticon handling
  - Indonesian language tokenization
  - Handling slang and informal language

### Model Architecture
We employed **BERT (Bidirectional Encoder Representations from Transformers)** for its superior understanding of context:

1. **Base Model**: IndoBERT (BERT pre-trained on Indonesian corpus)
2. **Fine-tuning**: Custom classification layers for sentiment
3. **Output**: Multi-class sentiment (Positive, Neutral, Negative)

### Key Features
- **Text Preprocessing**: Advanced NLP pipeline for Indonesian text
- **BERT Fine-tuning**: Transfer learning on domain-specific data
- **Multi-class Classification**: Positive, Neutral, Negative sentiments
- **Real-time Analysis**: API endpoint for live sentiment detection
- **Aspect-Based Analysis**: Identify specific features mentioned
- **Trend Visualization**: Track sentiment over time

## Code Example

```python
from transformers import AutoTokenizer, AutoModelForSequenceClassification
import torch

# Load fine-tuned model
model = AutoModelForSequenceClassification.from_pretrained('./bca-sentiment-model')
tokenizer = AutoTokenizer.from_pretrained('indolem/indobert-base-uncased')

def analyze_sentiment(text):
    inputs = tokenizer(text, return_tensors="pt", padding=True, truncation=True)
    outputs = model(**inputs)
    predictions = torch.nn.functional.softmax(outputs.logits, dim=-1)
    
    sentiment_map = {0: "Negative", 1: "Neutral", 2: "Positive"}
    predicted_class = torch.argmax(predictions).item()
    confidence = predictions[0][predicted_class].item()
    
    return {
        "sentiment": sentiment_map[predicted_class],
        "confidence": confidence
    }

# Example usage
review = "Aplikasi BCA sangat mudah digunakan dan cepat!"
result = analyze_sentiment(review)
print(f"Sentiment: {result['sentiment']} (Confidence: {result['confidence']:.2%})")
```
